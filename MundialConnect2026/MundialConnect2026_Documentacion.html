Kick Off y Primera Entrega - MundialConnect 2026

Nombre de la aplicación

MundialConnect 2026

Objetivo de la aplicación

Una plataforma web para aficionados del Mundial de Fútbol 2026, con registro y login de usuarios. Permite acceder a un panel con noticias, resultados y seguimiento de equipos favoritos.

Funcionalidades a desarrollar

  • Registro y login de usuarios.
  • Dashboard con noticias y resultados del Mundial 2026.
  • Gestión de perfiles (equipo favorito, avatar).
  • Amigos entre usuarios registrados.

Diseño de pantallas

  • Login: Formulario email y contraseña, botón de acceso.
  • Registro: Formulario para crear cuenta nueva con nombre, email y contraseña.
  • Dashboard: Noticias, resultados y panel de seguimiento de equipos favoritos.

Diagrama relacional de la base de datos

La base de datos incluye las siguientes tablas:

  • Usuarios: id (PK), nombre, email, password_hash, fecha_registro
  • Perfiles: id (PK), usuario_id (FK), equipo_favorito, avatar_url
  • Amistades: id (PK), usuario1 (FK), usuario2 (FK), estado, fecha_solicitud

Diagrama incluido en er_diagram.svg si quieres visualizarlo gráficamente.
