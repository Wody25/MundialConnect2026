Segunda Entrega - MundialConnect 2026

Nombre de la aplicación

MundialConnect 2026

Objetivo de la aplicación

Plataforma web para aficionados del Mundial 2026 en México, EE.UU. y Canadá, con registro y login de usuarios. Permite acceder a un dashboard con noticias, resultados, seguimiento de equipos favoritos y perfiles de usuario.

Funcionalidades actuales y desarrolladas

  • Registro y login de usuarios con seguridad de contraseña.
  • Dashboard con noticias del Mundial 2026 estilo foro.
  • Selección y seguimiento de equipos favoritos con visualización de próximos partidos y resultados (en desarrollo).
  • Perfil de usuario mostrando información del usuario y sus equipos favoritos.
  • Botones de navegación: Logout y Perfil.
  • Gestión de favoritos en la base de datos por usuario.
  • Animaciones de banderas y texto destacado en el dashboard.

Futura funcionalidad - Amistades

Se implementará un sistema de amigos dentro de la plataforma que permitirá a los usuarios:

  • Enviar solicitudes de amistad a otros usuarios registrados.
  • Aceptar o rechazar solicitudes de amistad recibidas.
  • Visualizar un listado de amigos aprobados.
  • Ver publicaciones o noticias compartidas por los amigos, integradas en el dashboard.
  • Integración con la sección de noticias para que los usuarios puedan comentar o reaccionar a publicaciones de amigos (en desarrollo).

Esta funcionalidad estará conectada con la tabla Amistades de la base de datos.

Diseño de pantallas y funcionalidades

  • Login: Email y contraseña, botón de acceso.
  • Registro: Nombre, email, contraseña, botón para crear cuenta.
  • Dashboard: Noticias estilo foro, animación de banderas, panel de equipos favoritos, seguimiento de partidos (en desarrollo), botón de Perfil, botón de Logout.
  • Perfil: Información del usuario desde la tabla de perfiles, lista de equipos favoritos, botón para regresar al Dashboard.

Diagrama relacional de la base de datos

La base de datos incluye las siguientes tablas:

  • Usuarios: id (PK), nombre, email, password_hash, fecha_registro
  • Perfiles: id (PK), usuario_id (FK), equipo_favorito, avatar_url
  • Amistades: id (PK), usuario1 (FK), usuario2 (FK), estado, fecha_solicitud
  • Favoritos: id (PK), usuario_id (FK), equipo_id (FK), fecha_seleccion
  • Equipos: id (PK), nombre, bandera_url
